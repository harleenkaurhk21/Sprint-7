# React Onboarding Wizard

A modern multi-step onboarding wizard built using React and Vite. The application collects user information in multiple steps with real-time validation, progress tracking, and a responsive UI.

## 🚀 Live Demo

https://sprint-7-taupe.vercel.app/

---

## Features

- Multi-Step Form (3 Steps)
- Personal Information
- Account Details
- Review & Submit
- Real-Time Validation
- Progress Bar
- Show/Hide Password
- Parent State Management
- Data Persistence
- Success Screen
- Responsive Design
- React Hook Form
- Zod Validation

---

## Tech Stack

- React.js
- Vite
- React Hook Form
- Zod
- React Icons
- CSS3
